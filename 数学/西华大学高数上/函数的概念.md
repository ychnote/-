# 集合与运算  
   - 集合：具有某种属性对的事物的全体称为集合， 一般用大写的A、B、C表示; 例如： a  $\in$ A , a $\notin$ A  

   - 集合分:有限集和无限集
    A = {a1, a2, ''''',an} 称为有限集
    M = {x| x 所具有的特征}称为无限集

##### 直集
   设A，B是任意两个集合，任取a $\in$ A, b $\in$ B ,则a,b组成了一对有序对（a,b）
    记作 A x B 
       A x B = {(a,b)| a $\in$ A , b $\in$ B}

   例：  
      A = {0，1 }  B = {-2，2}
      A x B
      解： A x B = {(0, -2), (0, 2), (1, -2), (1, 2)}
